# Webpack Starter
[![license](https://img.shields.io/github/license/mashape/apistatus.svg?maxAge=2592000)]()

A plain and opinionated boilerplate for the frontend.

Includes
---------------
- Webpack
- jQuery
- html-webpack-plugin
- css-loader
- style-loader

Installation
---------------
- Clone the repo
- yarn install
- Run ```npm run dev``` or ```npm run prod```
